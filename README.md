<h1>Credit Card Customer & Transaction Analysis Dashboard</h1>

<h4>This project analyzes credit card customer demographics, spending behavior, transaction patterns, and revenue metrics using SQL, Python, and Power BI. The dataset was cleaned and preprocessed using Python (Pandas, NumPy), stored and queried using SQL, and visualized through an interactive Power BI dashboard.</h4>
